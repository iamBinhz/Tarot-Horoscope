# ConclusionBox Content Review Skill

## When to Use
After a Tarot reading is generated, to evaluate the quality and coherence
of the entire `div#conclusionBox` (Reading Synthesis) output.

## Pre-Condition
- Question MUST be in `#qInput`
- Correct model MUST be selected
- Cards MUST be flipped and reading panel visible

## Target Structure
The conclusionBox (`div.conclusion-box#conclusionBox`) contains:
```
├── .conclusion-title         → "☽ Reading Synthesis ☽" / "☽ Tổng Hợp Bài Đọc ☽"
├── .synth-section
│   └── .synth-opening        → Domain + model-aware opening statement
├── .synth-section
│   ├── .synth-section-title  → "Narrative & Connections"
│   └── .synth-story          → generateUnifiedNarrative() output (6 parts)
├── .synth-section
│   ├── .synth-section-title  → "Element Analysis"
│   └── .synth-elements       → analyzeElements() + analyzeElementFlow()
├── .synth-section
│   ├── .synth-section-title  → "Path Forward"
│   └── .synth-path           → generateIntegratedActions() steps (ol > li)
└── .synth-badges-expanded
    ├── .synth-badge-card     → Energy badge (icon + type + description)
    ├── .synth-badge-card     → Confidence badge (level + reasons)
    └── .synth-badge-card     → Reading Tone badge (icon + tone + description)
```

## Review Criteria

### 1. Opening Statement (8%)
- [ ] Matches selected question model (general/problem/solution/crossroads)
- [ ] Matches detected domain (career/love/finance/health/family/personal)
- [ ] Sets appropriate tone for the reading
- [ ] Not generic — feels specific to the context

### 2. Narrative & Connections — Story Quality (22%)
Generated by `generateUnifiedNarrative()`, 6 parts:
- **P1** (Past card sentence): Accurately reflects card's past meaning?
- **P2** (Past↔Present interaction): Card pair interaction relevant?
- **P3** (Present card sentence): Addresses current situation?
- **P4** (Present↔Future interaction): Logical transition?
- **P5** (Future card sentence): Grounded, non-sensational?
- **P6** (Arc closing): Synthesizes the 3-card journey?

Evaluate:
- [ ] All 6 parts flow as a coherent story
- [ ] Reversed cards properly reflected
- [ ] CARD_INTERACTIONS_MAP entries contextually appropriate
- [ ] Prose is clear, evocative, free of awkward phrasing
- [ ] No contradictions between adjacent parts

### 3. Element Analysis (12%)
- [ ] Dominant elements correctly identified
- [ ] Missing elements correctly noted
- [ ] Arcana ratio text accurate
- [ ] Element flow arc narrative matches actual element sequence
- [ ] Element conflict narratives appropriate
- [ ] Analysis doesn't contradict the narrative section

### 4. Path Forward — Action Steps (18%)
Generated by `generateIntegratedActions()`:
- [ ] Step 1 (Past lesson): Reflective, connects to past position
- [ ] Step 2 (Present action): Actionable, relevant to "now"
- [ ] Step 3 (Future prep): Practical preparation advice
- [ ] Step 4 (Integrated advice): Synthesizes cards + domain + model
- [ ] Steps specific enough to be useful (not platitudes)
- [ ] Steps logically progress past → present → future

### 5. Context Relevance (15%)
- [ ] Opening references the subject's domain?
- [ ] Narrative relates to the subject's situation (bối cảnh)?
- [ ] Action steps address the subject's ACTUAL question?
- [ ] Would this reading feel personally relevant to the Test Subject?
- [ ] Domain detection (`extractDomain()`) produced correct domain?

### 6. Energy Assessment (8%)
- [ ] `determineEnergy()` result makes sense for drawn cards
- [ ] Energy description matches actual card combination
- [ ] Not contradicted by narrative or tone

### 7. Confidence Level (8%)
- [ ] `determineConfidence()` returns appropriate level
- [ ] Listed reasons valid and verifiable
- [ ] Edge cases: all reversed, no dominant element, future reversed

### 8. Reading Tone (9%)
- [ ] `determineReadingTone()` matches overall feel
- [ ] Tone consistent with energy level
- [ ] Tone doesn't contradict narrative's emotional arc

## Scoring
- Each criterion: 0 (fail) to 5 (excellent)
- Total score out of 100 (weighted per above percentages)
- Threshold: ≥ 70 = PASS, 50-69 = MARGINAL, < 50 = FAIL

## Special Focus: Future Prediction Quality
1. Prediction grounded in card's established meaning?
2. Avoids overpromising or fear-mongering?
3. Clear logical path from present → future?
4. Action steps practical and achievable?
5. Tone matches gravity of prediction?

## Anti-Patterns
- Don't penalize randomness of card draws
- Don't expect specific literary style — evaluate clarity and coherence
- Don't ignore Vietnamese output — both languages must be equally good
- Don't accept generic phrases as quality prose
