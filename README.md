# The Celestial Sanctum — Tarot & Horoscope

A mystical web app for tarot card reading and Vietnamese astrology (Tử Vi Đẩu Số). No build tools, no frameworks — just open in a browser.

## What It Does

### Tarot Reading (`tarot-example.html`)
Draw a three-card spread (Past / Present / Future) from a full 78-card deck. Each reading includes upright and reversed interpretations, elemental analysis across the spread, card interaction patterns, and a synthesized conclusion — all bilingual (English / Vietnamese).

### Tử Vi Natal Chart (`natal_chart.html`)
Enter your birth date, time, and gender to generate a traditional Vietnamese Purple Star Astrology chart. The app converts your Gregorian birthdate to the lunar calendar, places 14 main stars and auxiliary stars across 12 palaces, and produces a full personality and life-path interpretation including:
- Star combinations per palace
- Tứ Hóa (Four Transformations)
- Ngũ Hành (Five Elements) analysis
- Đại Hạn (Major Life Periods)
- Cách Cục pattern recognition

### Landing Page (`index.html`)
Hub page linking to all sections with an animated starfield and celestial theme.

## Tech

- Vanilla HTML / CSS / JavaScript — no dependencies
- Google Fonts via CDN
- Canvas API for animations
- Open any `.html` directly in a browser

## Quick Start

```
git clone https://github.com/iamBinhz/Tarot-Horoscope.git
```
